Three Steps to Your First Analysis

1

Open Project in Cursor IDE

Launch Cursor and open the numbers_eeg_source directory.

In Cursor, press Ctrl + K then Ctrl + O

Or: File ‚Üí Open Folder

Navigate to where you cloned the repository

In Cursor, press Cmd + K then Cmd + O

Or: File ‚Üí Open Folder

Navigate to where you cloned the repository

In Cursor, press Ctrl + K then Ctrl + O

Or: File ‚Üí Open Folder

Navigate to where you cloned the repository

2

Activate Python Environment

Open Cursor's integrated terminal and activate the environment.

Press Ctrl + ` to open the terminal, then:

conda activate numbers_eeg_source

Press Cmd + ` to open the terminal, then:

conda activate numbers_eeg_source

Press Ctrl + ` to open the terminal, then:

conda activate numbers_eeg_source

Verify Environment

You should see (numbers_eeg_source) at the beginning of your terminal prompt.

3

Run Your First Analysis

Execute the example analysis pipeline using the provided scripts.

Option A: Using the Batch Script (Easy)

cd new_user
run_analysis.bat

Option B: Direct Python Command

python -m code.run_full_analysis_pipeline --config new_user/examples/sensor_13_31.yaml --accuracy all
Tip:

The batch script automatically activates the environment and runs the analysis. Use Option A if you want the simplest approach!

Option A: Using the Shell Script (Easy)

cd new_user
./run_analysis.sh
First Time?

If you get a "permission denied" error, make the script executable:

chmod +x run_analysis.sh

Option B: Direct Python Command

python -m code.run_full_analysis_pipeline --config new_user/examples/sensor_13_31.yaml --accuracy all

What to Expect

  • Analysis will take 5-15 minutes depending on your computer
  • You'll see progress messages in the terminal
  • Results will be saved in the results/ directory
  • Look for a success message when complete
